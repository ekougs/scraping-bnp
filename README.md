# Scraping BNP Demo App
> Simple scraping of BNP demo app

## Install dependencies

```sh
$ npm install
```
Or
```sh
$ yarn install
```

## Demo
```sh
$ npm start 123 123456
```
First parameter is user ID and second, user password

## Test
```sh
$ npm test
```
